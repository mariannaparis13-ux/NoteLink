Notes

Students can upload notes when requested by other students. These notes will be available fo rall students to access. Notes are organized by grade and subject. The subjects that are taught seperatly for different mediums (english, sinhala or medium) have sepperate notes. However the subjects that are taught commonly for all mediums will share the same notes.

How to view notes

  1. In the Homepage log in using the admission number.
  2. Click on the 'Notes' button
  3. Select your grade and medium using the dropdown boxes and login.
  4. Click on the subject that you want to view
  5. Click on the lesson that you want to view.

How to delete notes

Follow steps 1-4 as mentioned above, and click the 'Delete file' button next to the note you want to delete.

Requests

A feature that allows students to request notes from other students who have them.

How to add a request

  1. In the Homepage log in using the admission number.
  2. Click on the 'Requests' button
  3. Select your grade and medium using the dropdown boxes and login.
  4. Click on the subject that you want to view
  5. First, check through the list of existing requeststo see if someone else has already requested the note you want. If not, type your request in the textbox and click 'Add request'. Pls make sure to only use numbers, letters and brackets.
  6. If your request is successful, you will see a thank you message and automatically be sent back to the request page where you can see your request.

How to add notes

  1. Follow steps 1-4 above.
  2. Click on the 'Upload Notes' button next to the request you want to help with.
  3. Check if the grade, subject and lesson name are correct.
  4. Click on 'Choose file', and select the note file from your device.
  5. Check if the correct file was selected and click 'Upload Note'. (Pls refrain from directly uploading image files as this will take a lot of space. We suggest to use 'Adobe Scan' or a similar app.)
  6. If your upload is successful you will see a thank you message and automatically be sent back to the request page where you can see that the request you fulfuiled is now removed.
